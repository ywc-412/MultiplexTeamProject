# Multiplex Team Project

## 시스템 개요

#### 영화관 관리 시스템은 영화를 등록하고 공지사항 게시판 등을 관리할 수 있고,
#### 회원들은 영화를 예매하거나 정보 및 리뷰 등을 공유할 수 있는 환경을 제공하는 시스템이다.
