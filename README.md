# Multiplex Team Project

* 영화 등록 및 예매, 게시판 이용과 기프티콘 구매가 가능한 사이트
* 참여자 : 박진주, 오한나, 이영주, 이희홍, 최영우
* 맡은역할 : 기프티콘, 내 기프티콘, 공지사항 게시판
